Apakah Kotlin itu?

  • Kotlin adalah bahasa pemrograman baru yang dikembangkan JetBrains.
  • Pertama muncul pada tahun 2011 ketika JetBrains meluncurkan projek “Kotlin”
    • Kotlin adalah bahasa yang Open source.
    • Kotlin menjadi bahasa resmi pengembangan Android.
  • Kotlin mendukung pemrograman prosedural dengan penggunaan function

Keuntungan Bahasa Kotlin

  • Kotlin dikompilasi ke bytecode JVM atau JavaScript 
  • Kotlin dapat menggunakan framework dan pustaka Java  
  • Kotlin dapat dipelajari dengan mudah
  • Ada layanan konversi dari Java ke Kotlin
  • Kotlin’s null-safety !!

NullPointerExceptions

      val   name: String = null   // tries to assign null, won’t  compile.
  
      fun   getName() : String = null   // tries to return null, won’t  compile.

Fitur unggulan bahasa Kotlin

  • Ringkas
    • Secara drastis mengurangi jumlah kode boilerplate ("= template").
  • Aman
    • Mengindari kesalahan pada Class seperti pengecualian pointer null.
  • Interoperabilitas
    • Memanfaatkan pustaka yang ada untuk JVM, Android, and the browser.
  • Tool yang ramah
    • Silahkan pilih Java IDE atau mengembangkan dari command line.
  • Tidak ada dampak performansi, run time sangat kecil (< 1MiB).

Perhatikan contoh berikut

public class Address {

   private String city;
   private Country country;

   public Address(String city, Country country) {
       this.city = city;
       this.country = country;
   }

   public String getCity() {
       return city;
   }
   public void setCity(String city) {
       this.city = city;
   }

   public Country getCountry() {
       return country;
   }
   public void setCountry(Country country) {
       this.country = country;
   }
}

data class Address(var city:String, var country:Country)
